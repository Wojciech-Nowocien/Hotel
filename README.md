# Hotel

Konsolowa aplikacja symulująca działanie hotelu.
Projekt napisany w języku `Java`, będący prostą symulacją podstawowych operacji hotelowych.

## Opis
Ta aplikacja stanowi symulację działania hotelu w konsoli. Umożliwia m.in.:
- rezerwowanie pokoi (i anulowanie tych rezerwacji),
- zarządzanie pokojami,
- wyświetlanie historii pokoju,

## Jak korzystać z aplikacji?

Aplikacja jest niezwykle intuicyjna. Wystarczy postępować zgodnie z komunikatami. Aby wybrać jakąś opcję wystarczy podać
odpowiadający jej numer.
